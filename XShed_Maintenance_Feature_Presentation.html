👥 Features by Portal
👤 Customer Portal
  • Maintenance + Schedule buttons on DELIVERED orders only
  • Schedule Extra Cleaning (locked service, choose type)
  • Fee auto-calculated from DB (service × type matrix)
  • Date picker + time slot selection
  • Optional customer notes field
  • Maintenance History modal (9 columns)
  • Pagination on history table
  • Mobile card view for history
  • Cancel pending maintenance requests
  • Success / cancel toast notifications
  • Bills & Payments shows maintenance charges
  • Damage Protection replaced by Service Type + Type

  • Amount Due = actual service fee charged

  • Remarks field for delivery company notes
  • Cancel pending maintenance requests
  • Extra Cleaning payment dialog (mark as Done Cleaning)
  • Creates PaymentTransaction + MonthlyBilling on done
  • Order list auto-refreshes after done / cancel
  • Success / cancel toast notifications
  • Mobile responsive card view
  • Pagination on history
🏢 Franchisee / Admin Portal
  • Bills & Payments shows maintenance payment rows
  • Damage Protection replaced by Service Type + Type
  • Due Date / Job Name / PO# show N/A for maintenance
  • Amount Due = actual service fee charged
  • Works for both getAllMonthlyBillingByUserId and ByClientId queries
  • isMaintenance flag on all billing responses
  • maintenanceServiceType + maintenanceTypeDesc fields returned

💳 Bills & Payments Integration

When a delivery company marks Extra Cleaning as Done, the following chain executes automatically:

donecleaning AJAXManageMaintenanceAction
→
Update maintenancestatus = Paid/Completed
completionTime recorded
→
Create PaymentTransactionuser = customer's user
is_cancelled = NULL
is_refunded = NULL
amountPaid = service fee
→
Create MaintenancePaymentTransactionlinks maintenance ↔ payment
→
Create MonthlyBillingorder_details_id + payment_transaction_id
→ appears in Bills & Payments
